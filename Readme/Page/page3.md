# 3. 실시간 위치 기반 트레이너 및 프로모션 검색

## 페이지 소개
* 실시간 위치 기반으로 트레이너 및 프로모션을 확인할 수 있는 페이지입니다.
<br><br>
## 구동원리
* GPS로 실시간 내 위치를 파악하여 지도에 트레이너와 프로모션을 뿌려줌(Daum Map Api)
* 회원이 직접 선택한 위치를 지도에 뿌려줌(공공데이터 오픈 API(부동산 개방데이터의 법정동조회서비스))
