# 2. 메인

## 페이지 소개
* 인기코스 및 운동 종목별 프로모션을 확인할 수 있는 페이지입니다.
* 실시간 위치 기반으로 트레이너 및 프로모션을 확인할 수 있는 페이지입니다.
<br><br>
## 구동원리
* 일정한 시간이 지나면 운동종목 사진이 슬라이드로 넘어가게 함(Swiper)
* 스크롤링을 통해 종목별 프로모션을 추가로 가져오게 함
