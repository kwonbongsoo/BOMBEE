# 13. Diet Management

## 페이지 소개
* 일반 회원/트레이너의 진행 중이 프로모션에서 식단 관리를 받을 수 있는 페이지입니다.
<br><br>
## 구동원리
* 일반 회원은 당일 식단 사진만을 blue-imp-fileupload를 이용하여 업로드 및 변경함
* moment.js를 통해 일주일, 일년 단위의 가공된 날짜 데이터를 가져와 뿌려줌
